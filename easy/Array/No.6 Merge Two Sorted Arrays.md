### Description
Merge two given sorted integer array A and B into a new sorted integer array.

### Example
Example 1:<br>
	Input:  A=[1], B=[1]<br>
	Output: [1,1]<br>
	Explanation: <br>
	return array merged.<br>

Example 2:<br>
	Input:  A=[1,2,3,4], B=[2,4,5,6]<br>
	Output: [1,2,2,3,4,4,5,6]<br>
	Explanation: <br>
	return array merged.<br>
