### Description
Given an integer array, find a subarray where the sum of numbers is zero. <br>
Your code should return the index of the first number and the index of the last number.
### Example
Example 1:
	Input:  [-3, 1, 2, -3, 4]
	Output: [0, 2] or [1, 3].
	
	Explanation:
	return anyone that the sum is 0.

  Example 2:
	Input:  [-3, 1, -4, 2, -3, 4]
	Output: [1,5]
